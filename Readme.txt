This is the project of CMT202 Distributed and cloud computing.
To correctly run this program, you need pipenv and Pyro4 in your enviroment.
After that, using command "pipenv run python -m Pyro4.naming" to start naming service
Then, "pipenv run python library.py" need to be used to start a server
Finally, "pipenv run python staff.py" will be used to start a client
